# 信源档案

记录各信息来源的特点、可访问性、可靠度评估。

## 国际新闻

### AP News（美联社）
- **网址：** apnews.com
- **访问：** ✅ 可直接 web_fetch
- **特点：** 事实报道为主，不评论不站队，业界标杆
- **可靠度：** ⭐⭐⭐⭐⭐

### NPR（美国国家公共电台）
- **网址：** npr.org
- **访问：** ✅ 可直接 web_fetch
- **特点：** 深度报道+评论，偏自由派立场，事实扎实
- **可靠度：** ⭐⭐⭐⭐

### UN News（联合国新闻）
- **网址：** news.un.org
- **访问：** ✅ 可直接 web_fetch
- **特点：** 全球治理、人道危机、气候、核不扩散
- **可靠度：** ⭐⭐⭐⭐⭐（视角偏国际主义）

### Reuters（路透社）
- **网址：** reuters.com
- **访问：** ❌ 连接失败（国内服务器限制）
- **备用：** 通过 Bing 搜索间接获取

### BBC News
- **网址：** bbc.com/news
- **访问：** ❌ RSS 连接失败
- **备用：** 通过 Bing 搜索间接获取

### Google News
- **网址：** news.google.com
- **访问：** ❌ 连接失败
- **备用：** 无

## 中文平台

### 百度热搜
- **网址：** top.baidu.com
- **访问：** ✅ 可直接 web_fetch
- **特点：** 按搜索指数排名，反映全网主动搜索行为
- **局限：** 仅限中国网民搜索数据，不代表全球

### 今日热榜（tophub.today）
- **网址：** tophub.today
- **访问：** ✅ 可直接 web_fetch（慢但可用）
- **特点：** 聚合多平台（微博/知乎/百度/抖音/B站/贴吧）
- **局限：** 各平台数据质量参差

### 知乎热榜
- **网址：** zhihu.com/hot
- **访问：** ❌ 403 被拦
- **备用：** 通过 tophub.today 间接获取

### 微博热搜
- **网址：** weibo.com/ajax/side/hotSearch
- **访问：** ❌ API 403 禁止
- **备用：** 通过 tophub.today 间接获取

## 社交平台

### X.com（原 Twitter）
- **网址：** x.com
- **访问：** ❌ 完全被墙（直接/Nitter/Bing 搜 site 均失败）
- **备用：** 需海外节点

## 搜索引擎

### Bing（cn.bing.com）
- **访问：** ✅ 可用（国内版）
- **特点：** 搜索结果可抓取，但新闻搜索会重定向到首页
- **技巧：** 用普通搜索而非新闻搜索

### Google
- **访问：** ❌ 被拦（private IP blocked）

## 教训
- 单一信源不能代表「全网」，必须多源交叉验证
- 百度热搜=中国网民搜索热度，不等于全球
- 抓取前先确认网站是否可访问，绕不过的就换路
