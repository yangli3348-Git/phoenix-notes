## 技术调研

### 阶段一：视频发布到抖音 ✅（已完成）

通过抖音开放平台的 **H5 分享能力** 实现：

1. 注册抖音开放平台开发者，创建**网站应用**
2. 申请开通「发布内容至抖音」权限（scope: `h5.share` + `open.get.ticket`）
3. 用户在你的网站上传视频（128MB 上限，mp4/mov）
4. 后端生成签名 schema，前端唤起抖音 APP 发布（deep link: `snssdk1128://`）
5. 依赖用户手机安装抖音 APP

**关键文档路径：**
- SDK 概述：`//developer.open-douyin.com/docs/resource/zh-CN/dop/develop/sdk/mobile-app/sdk`
- OpenAPI 列表：`//developer.open-douyin.com/docs/resource/zh-CN/dop/develop/openapi/list`
- Webhooks：`//developer.open-douyin.com/docs/resource/zh-CN/dop/develop/webhooks`

### 阶段二：追踪视频传播数据 ✅（有条件）

通过抖音开放平台的 **「视频互动数据」能力** 实现，需要**额外申请权限**。

**能力详情页：** `//developer.open-douyin.com/capacity-center-page/capacity-detail/7448483112163590181`

**能力中心入口：** `//developer.open-douyin.com/capacity-center-page` — 筛选「网站应用/移动应用」+「数据/短视频」分类，藏着不少不在常规 OpenAPI 文档中的能力。

**可获取的数据：**

| 接口 | 返回 | 文档路径 |
|---|---|
| 获取视频基础数据 | 平均播放时长、点赞数、评论数、播放数、分享数 | 能力中心查看文档 |
| 获取视频点赞数据 | 点赞数 | 能力中心查看文档 |
| 获取视频评论数据 | 评论数 | 能力中心查看文档 |
| 获取视频播放数据 | 播放数 | 能力中心查看文档 |
| 获取视频分享数据 | 分享数 | 能力中心查看文档 |

**⚠️ 准入条件与限制：**
- **仅限企业、党政和事业单位类型开发者**申请（个人开发者不可用）
- 用户首次授权后，需要**第二天**才会产生数据（非实时）
- 不支持沙盒环境
- 审核 2-3 个工作日
- 不可用于搭建对外售卖的「数据服务类型」应用
- 滥用会被封禁
