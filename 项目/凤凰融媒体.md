### 中国平台逐平台深搜结果（2026-04-26）

每个平台单独搜索后确认：**没有现成的中国平台官方API Python SDK适配器可直接拿来用。**

| 平台 | 搜索结果 | 情况 |
|---|---|---|

| 抖音 | wulinyun/java-sdk-douyin (10⭐) | Java/Python/PHP多语言版，2022停更，不完整 |
| 快手 | 0结果 | 连非官方SDK都没有 |
| B站 | Nemo2011/bilibili-api | 社区维护的非官方库，偏数据采集 |
| 微博 | 0结果 | OpenAPI已逐步下线，最弱一环 |
| 微信视频号 | 0结果 | 无公开OpenAPI用于发布 |
| 综合 | dreammis/social-auto-upload (10.2k⭐) | 唯一有完整中国覆盖，但全是Playwright浏览器自动化 |

| **Nemo2011/bilibili-api** | B站API接口定义方式参考 |

### 结论
- 没有现成的中国社媒平台适配器集成在一起的开源项目
- **BrightBean 提供框架 + 海外平台适配器**，我们写中国平台适配器
- social-auto-upload 的浏览器自动化代码可参考上传流程细节，但企业级发布需转写为官方API适配器
- 微博是最大缺口，所有开源项目对微博支持都最弱

## Demo 原型（2026-04-26）

为给台长展示，搭建了「凤凰卫视融媒体内容中台」可交互Demo大屏。

### 技术栈
- **前端**：Vue 3 + ECharts 5（柱状图/折线图/世界地图飞线）
- **后端**：FastAPI，模拟数据API接口
- **端口**：8502（前端）/ 8510（后端）
- **位置**：`feihuang-demo/`

### 效果
- 深色科技蓝大屏风格，凤凰logo + 品牌色
- 5大核心指标卡片（今日分发/播放/点赞/评论/分享）
- 全球飞线地图：香港总部→20个记者站城市（散点+飞线动画）
- 国内vs海外渠道分布对比（进度条）
- 各平台播放量对比柱状图
- 近7天播放趋势折线图
- 近期内容分发状态列表（凤凰真实栏目名）
- 数据每30秒自动刷新，飞线每10秒重绘

### 启动
```
后端：cd feihuang-demo/backend && uvicorn main:app --host 0.0.0.0 --port 8510
前端：cd feihuang-demo/frontend && npx vite --port 8502 --host 0.0.0.0
访问：http://服务器IP:8502
```
