# 工具能力（本服务器实测）

> 记录这台服务器（国内Linux）上各工具的可用性。更新于2026-04-25。

## 网络访问

| 目标 | 状态 | 方式 |
|------|:----:|------|
| **GitHub** (网页/API/CLI) | ✅ 全通 | web_fetch / curl / git |
| **arXiv** | ✅ 可 | web_fetch 抓摘要+PDF |
| **PubMed** | ✅ 可 | web_fetch |
| **Bing搜索** | ✅ 可 | web_fetch (cn.bing.com) |
| **百度热搜** | ✅ 可 | web_fetch (top.baidu.com) |
| **今日热榜** | ✅ 可 | web_fetch (tophub.today，较慢) |
| **AP News** | ✅ 可 | web_fetch |
| **NPR** | ✅ 可 | web_fetch |
| **UN News** | ✅ 可 | web_fetch |
| **Semantic Scholar** | ⚠️ 半残 | 首页可用，搜索结果JS渲染抓不全 |
| **Google Scholar** | ❌ 被墙 | 不可用 |
| **X.com (Twitter)** | ❌ 被墙 | 直接/Nitter/Bing均失败 |
| **Reuters / BBC / Google News** | ❌ 被墙 | 连接失败 |

## 信息获取方法

### 搜索学术论文
1. Bing搜索关键词 → 找到 arXiv/PubMed 等链接
2. 直接抓取对应论文页面

### 搜索新闻
1. Bing搜索关键词 → 找到新闻链接
2. 直接抓取新闻原文（AP、NPR、UN News 可用）
3. 或爬百度热搜获取中国热门话题
4. 多源交叉验证，避免单一信源偏差

## 工具限制

### web_fetch
- 不支持JS动态渲染页面
- 可能被403拦截（知乎、微博API、部分反爬）
- 可设 maxChars 控制返回长度
- 支持 RSS/XML

### browser（浏览器自动化）
- 可处理JS渲染页面（备用方案）
- 需要浏览器运行环境

### git CLI
- 完整功能可用：clone / pull / push / log / ls-remote
- 可配合 GitHub API 使用
